Scrivi un programma in linguaggio C che\+:

Francesco Di Lena, Classe 4F

\begin{DoxyDate}{Date}
02/12/2021 
\end{DoxyDate}
\begin{DoxyVersion}{Version}
1.\+0
\begin{DoxyItemize}
\item genera due processi padre/figlio;
\item il processo padre legge il nome e cognome dell’utente e lo trasforma con l’iniziale maiuscola e le altre lettere minuscole (cioè per esempio inserendo “rossi MARIO” produce “\+Rossi Mario”);
\item il processo padre scrive il nome così trasformato in un file di testo aperto prima di eseguire la fork;
\item dopo aver completato la scrittura del nome, chiude e rimuove il file creato;
\item contemporaneamente il processo figlio attende 5 secondi e stampa il contenuto del file appena scritto dal pro-\/ cesso padre convertendo le parole in maiuscolo. 
\end{DoxyItemize}
\end{DoxyVersion}
